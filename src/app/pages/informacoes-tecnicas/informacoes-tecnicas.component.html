Fonte

  • Os dados de evolução diária do COVID-19 no município de Catanduva/SP (de casos notificados, confirmados, suspeitos, descartados, bem como de curados, óbitos e de pacientes internados) são obtidos através do Boletins do Coronavírus: Prefeitura Municipal de Catanduva, divulgado diariamente no site oficial da Prefeitura de Catanduva/SP.
  • Os dados de Síndrome Respiratória Aguda Grave (SRAG) - testagem, de localização dos casos confirmados de COVID-19, entre outros, são obtidos através do openDataSus.

Notas

  • Transcrevemos os dados epidemiológicos, assim que divulgados pela Prefeitura, de forma manual: da imagem para um formato estruturado (tabular). Naturalmente, podem ocorrer erros humanos nessa transcrição. Caso note alguma divergência dos dados aqui publicados com os dados oficiais, por favor, nos escreva via redes sociais, e-mail, ou pelo Github.
